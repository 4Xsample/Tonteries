# Repositori de Tonteries
Benvingut al Repositori de Tonteries! Aquí trobaràs una recopilació de coses divertides, curioses i totalment inútils que m'han fet gràcia i crec que poden ser d'interès per a altres persones amb un sentit de l'humor similar al meu.
No esperis trobar-hi res útil.

## Com contribuir?
Si tens alguna altra idea de tonteria que creguis que mereix ser compartida amb el món, no dubtis a fer una contribució! Simplement fes un fork d'aquest repositori, afegeix la teva tonteria i envia una pull request. Estic obert a totes les idees estrafolàries i divertides!
